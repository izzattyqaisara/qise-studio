# QISE Studio Fullscreen Click Pages Website

This version removes the faux phone/browser border and makes the website fully maximised on the screen.

The homepage shows first. Other sections open when clicked from the navigation:
- Home
- About
- Services
- Portfolio
- Businesses
- Contact

## How to run
1. Open this folder in VS Code.
2. Right-click `index.html`.
3. Choose `Open with Live Server`.

## Files
- index.html
- styles.css
- script.js
